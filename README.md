# Git Buoy Homebrew tap

This tap installs the supported Apple Silicon build of
[Git Buoy](https://github.com/markg-05/git-buoy).

```sh
brew install markg-05/tap/git-buoy
```

Then point Git Buoy at a repository:

```sh
git-buoy path/to/repository
```

Git and an interactive terminal are required. GitHub observation is optional
and requires an authenticated `gh` executable.
